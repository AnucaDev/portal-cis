# Documentacion CIS2026

Esta carpeta centraliza documentacion no ejecutable del proyecto.

## Estructura

- `tecnica/`: documentacion tecnica general del proyecto.
- `guias/`: guias operativas y de uso.
- `incidencias/`: resolucion de errores y troubleshooting.
- `artifacts/`: materiales auxiliares (pptx/docx).

## Nota

Estos archivos no afectan al runtime del portal ni de la API.
